Random network model

G(N,L) Model: N labeled nodes are connected with L randomly placed links. In this model, the number of links is fixed, and the density of the network is fixed as well

G(N,p) Model: Each pair of N labeled nodes is connected with probability p. In this model, the number of links is variable
