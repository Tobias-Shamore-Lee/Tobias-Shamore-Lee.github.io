Degree distribution P(k): probability that a randomly chosen node has degree k.

Random network model

G(N,L) Model: N labeled nodes are connected with L randomly placed links. In this model, the number of links is fixed, and the density of the network is fixed as well

G(N,p) Model: Each pair of N labeled nodes is connected with probability p. In this model, the number of links is variable

Number of links in a random network,P(L),the probability to have exactly L links in a network of N nodes and probability p:

P(L)=kpL(1−p)N(N−1)2−LP(L)=kp^L(1-p)^{\frac{N(N-1)}{2}-L} P(L)=kpL(1−p)2N(N−1)​−L

k for Number of different ways we can choose L links among all potential links.
k=((N2)L)k=\binom{\binom{N}{2}}{L}k=(L(2N​)​)

Insights obtained:

  1. In real networks, degrees are orders of magnitude higher than ⟨k⟩.
  2. The spread in the degrees of real networks is much wider than expected in a random network. This difference is captured by the dispersion σk\sigma ^kσk
  3. In real networks the degrees vary far more widely than predicted by random network theory.
